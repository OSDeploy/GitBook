# RegAdd AeroLite Theme

{% file src="../../../.gitbook/assets/regadd-aerolite-theme.inf" %}

If you are using the Auto ExtraFiles option for creating WinPE, you can change the WinPE Theme from Aero to AeroLite in WinPE by adding this Driver.  With this theme, the Windows will have a flatter Title Bar and Title Bar Buttons.

![WinPE Aero Theme](../../../.gitbook/assets/2018-10-18_22-59-25.png)

![WinPE AeroLite Theme](../../../.gitbook/assets/2018-10-18_22-59-31.png)
